Instrucciones breves para ejecutar el proyecto:

1- Abre una terminal dentro de la carpeta del proyecto

2- Ejecuta este comando: para descargar las carpetas necesarias de Laravel: composer install

3-Configurar el archivo .env: Busca el archivo .env.example, cámbiale el nombre a .env

4-Dentro de la carpeta database, crea un archivo nuevo vacío llamado database.sqlite

5- Ejecuta estos dos comandos en orden:

php artisan key:generate

php artisan migrate:fresh --seed

6-Ejecuta el comando: php artisan serve
